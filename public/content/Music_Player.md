# Music Player

**Web-based Audio Streaming**

Developed a web-based music player with Cloudflare R2 CDN integration for efficient audio streaming and quick load times. Optimized frontend rendering for smooth playback.

## Tech Stack
- **Frontend**: React JS, Typescript
- **Storage**: Cloudflare R2 CDN
- **Key Features**: Efficient Streaming, Quick Load Times, Smooth Playback
