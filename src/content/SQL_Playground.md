# SQL Query Generator Application

**AI-Powered Query Platform**

Built a multi-database query platform where users can connect various databases and generate SQL queries from natural language using AI (Ollama). Reduced query-writing time by ~70% for target users.

## Tech Stack
- **Frontend**: React JS
- **Backend**: Express, Ollama
- **Database**: PostgreSQL
- **Key Features**: Natural Language to SQL, Multi-database support, AI Integration
