# API Maker

**No-Code REST API Builder**

Create REST API using web UI only. Users can create all REST endpoints and configure the application with different databases. Users can monitor, control, and customize any endpoints using the web UI.

## Tech Stack
- **Frontend**: React JS
- **Backend**: Express JS
- **Database**: MySQL
- **ORM**: Prisma
- **Key Features**: Dynamic Endpoint Creation, Database Configuration, Monitoring Dashboard
